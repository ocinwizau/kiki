# Daily QLD Deals Content Creation

## Task: Create daily deals content each morning (Brisbane time 6:30am)

### What to do:
1. Search for current Woolworths/Coles/Bunnings half price specials
2. Check for any breaking deals or news
3. Create 3-5 ready-to-post Facebook posts
4. Save to workspace as Daily_Deals_YYYY-MM-DD.md

### Posting times (recommended):
- 1st post: 6:30am (morning deals)
- 2nd post: 12pm (lunch deals)  
- 3rd post: 5pm (afterwork deals)

### Files to create:
